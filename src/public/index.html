Welcome to CW Chat

CW Chat is a real-time chat application in morse code with keyboard. It covers some rig-like features such as transmit / receive mode or channel based communication. Additionally, you can customise the frequency or speed if you opt for paddle key.
Happy Beeping!

If you have any issues, please reach out to the author via GitHub.

CW Chat

How to use

Receiving signal

When you are in receive mode, you can hear signals of others in this room but you cannot send any signals. User pill becomes pink in this mode.

Sending signal

When you are in transmit mode, your sinals will be sent to all users in this room but you cannot receive any signals. User pill becomes red in this mode.

Straight key
[space] beep

You can hold this key for the length of beep.

Paddle key
⬅️ dot ➡️ dash

You can hold this key for continuious dots or dashes.
